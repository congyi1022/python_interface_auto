## interface-demo01
这是一个最最基本的python实现自动化接口的脚本。
### 实现功能如下：
1. 对get/post接口进行封装，实现get/post请求；
2. 运行unittest框架，通过HTMLTestRunner生成测试报告；
3. 对报告进行邮件的发送。
